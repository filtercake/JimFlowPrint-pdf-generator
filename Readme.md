# Purpose

Gives a prototyping canvas with jade and sass.

## Set up

    $ git clone <this repo>
    $ npm install gulp -g
    $ npm install
    $ gulp

Go to: [http://localhost:3000/sth](http://localhost:3000/sth) to see an example prototype.
